void HurtIconsUI.AlertPlayer ( Vector3  direction,
Transform  t 
)

Thisa function takes a direction, and a transform representing the position the damage is coming from in order to make a decision on which alpha to set to one. The dot product is used here to determine where damage is coming from.

Parameters
directionThe direction the damage is coming from
tThe transform of the thing causing the damage. This is given because sometimes, like with an explosion, you need to know where a grenade is giving you damage, and sometimes the damage is coming from an enemy, not strictly whatever they're firing.
