Detailed Description

A component that creates a deformed mesh from a given one along the given spline segment. The source mesh will always be bended along the X axis. It can work on a cubic bezier curve or on any interval of a given spline. On the given interval, the mesh can be place with original scale, stretched, or repeated. The resulting mesh is stored in a MeshFilter component and automaticaly updated on the next update if the spline segment change.

enum SplineMesh.MeshBender.FillingMode
strong

The mode used by MeshBender to bend meshes on the interval.

Enumerator
Once 

In this mode, source mesh will be placed on the interval by preserving mesh scale. Vertices that are beyond interval end will be placed on the interval end.

Repeat 

In this mode, the mesh will be repeated to fill the interval, preserving mesh scale. This filling process will stop when the remaining space is not enough to place a whole mesh, leading to an empty interval.

StretchToInterval 

In this mode, the mesh is deformed along the X axis to fill exactly the interval.
