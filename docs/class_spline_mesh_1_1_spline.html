Detailed Description

A curved line made of oriented nodes. Each segment is a cubic BÃ©zier curve connected to spline nodes. It provides methods to get positions and tangent along the spline, specifying a distance or a ratio, plus the curve length. The spline and the nodes raise events each time something is changed.
